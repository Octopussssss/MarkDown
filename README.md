# 如何使用Mark Down

## 标题

大标题 #(space)My Project

第二大标题 ##(space)My Project  

最小标题 ###(space)My Project  

 ## 字体

斜体 * * 或 _ _ clt + I

粗体 ** ** 或 __ __ clt + B

## 其他

分割线 (* * *)(连着写)

删除线  ~~ ~~ (~~abc~~)

下划线  <u>abc </u>

[^jack han]: the best programmer on the world ↩

脚注 [^jack han]

Text 模式 >

> text

code 模式  clt + E ```  + 回车

```
git clone
git add
git commit
```

```
[jack han](http://www.hanfei.xyz/)
```

```
